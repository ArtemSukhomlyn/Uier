# uier

> A Vue.js project

## Prerequisites

* Install MongoDB as a service and open on port 27017

## Setup

**Client**
``` bash
cd client

# install dependencies
npm install
```

**Server**
``` bash
cd server

# install dependencies
npm install
```

## Start

**Client**
```bash
# serve with hot reload at localhost:8080
run-client.cmd
```

**Server**
```bash
# serve with hot reload at localhost:8081
run-server.cmd
```
